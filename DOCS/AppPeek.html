Class AppPeek
Implements the BOOSE peek command, which reads a value from an array at a specified index and assigns it to a variable.
Syntax
C#
    public  class AppPeek : ICommand
Remarks
Supported array types are integer and real (double) arrays. The command syntax is: peek <destVar> = <arrayName> <index>
