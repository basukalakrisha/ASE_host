Class AppPoke
Implements the BOOSE poke command, which assigns a value to a specific index of an existing array variable.
Syntax
C#
    public class AppPoke : ICommand
Remarks
Supported array types are integer and real (double) arrays. The command syntax is: poke <arrayName> <index> = <value>
