Class AppBoolean
Represents a boolean variable within the BOOSE language.
Syntax
C#
    public  class AppBoolean : Evaluation
Remarks
This command allows declaration and optional assignment of a boolean value. The syntax supported is: boolean <name> or boolean <name> = <expression>. Boolean values are internally synchronised with Value using 1 for true and 0 for false.
